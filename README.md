Exammer: get hooked on revision
An app to gamify and track progress on the completion of exam past paper questions

Quick setup guide:
- Clone the repo
- Run `npm install` to install dependencies
- Make a .env file based on the .env.example (you will need to provide your own Gemini API keys https://aistudio.google.com/api-keys)
- Run 'npm run build' to build the project
- Run 'npm run start' to start the server at http://localhost:8933
